## Bot and Glove for playing T-res
----------------
##### Required  ---
1. Arduino Uno or any other microcontroller 
2. Accelerometer (To notice the change in any direction )
3. IR sensor (To detect black and white colour)
4. Python library (pyautogui)
5. Arduino ide
